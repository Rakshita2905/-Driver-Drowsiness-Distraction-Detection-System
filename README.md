# 🚗 Driver Drowsiness & Distraction Detection System

A real-time computer vision system that detects driver drowsiness and distraction using facial landmarks and eye aspect ratio calculations.

## ✨ Features

- ✅ Real-time drowsiness detection using Eye Aspect Ratio (EAR)
- ✅ 3D facial landmark tracking (468 points)
- ✅ Adaptive thresholding for personalized detection
- ✅ Audio and visual alerts
- ✅ Data logging and session reports
- ✅ Low latency (30 FPS on standard hardware)

## 📋 Requirements

- Python 3.8 or higher
- Webcam
- Windows/Mac/Linux
